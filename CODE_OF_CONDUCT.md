# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Summary

All participants in this project (maintainers, contributors, users) are expected to:

- Be respectful and welcoming to newcomers.
- Give and gracefully accept constructive feedback.
- Focus on what is best for the community and the project.
- Assume good faith in others.
- Use welcoming and inclusive language.

Behavior that is disrespectful, disruptive or makes other contributors feel unwelcome is not acceptable and may result in temporary or permanent removal from the project spaces.

## Scope

This code applies to all project spaces: the repository, issues, pull requests, discussions, and any communication channel linked from this repository.

## Reporting

If you observe a problem, please contact the repository maintainer privately by opening a draft security advisory or by contacting them directly on GitHub. All reports will be reviewed and investigated promptly and fairly.
